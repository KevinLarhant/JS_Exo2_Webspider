Webspider
=================

Requirements :

- install nodejs
- run this 2 commands :
    - npm init
    - npm install request express lodash async winston --save


Lauch the app :

- Open a terminal
- Navigate to the Webspider directory
- type : 
    node spider.js 'the_URL_you_want_to_crawl'
  exemple :
    node spider.js http://www.google.fr
    
enjoy !
